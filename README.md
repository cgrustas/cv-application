# CV Builder

A React application that lets users create and preview a CV/resume in real-time. Built as part of [The Odin Project](https://www.theodinproject.com/).

## Skills Practiced

- **React Fundamentals** — Creating and composing components, understanding JSX syntax and how it differs from HTML
- **State Management** — Using `useState` to manage form data, handling arrays of objects in state, and understanding how state updates trigger re-renders
- **Props** — Passing data and callbacks between parent and child components to create reusable, customizable components
- **Controlled Components** - Determining which information should be controlled (with props), and which information should be uncontrolled (with state)
- **Conditional Rendering** — Showing different UI based on application state (edit mode vs. preview mode)
- **Rendering Lists** — Mapping over arrays to render multiple components, using unique keys to track items
- **Project Structure** — Organizing components and styles in a Vite + React project
- **Deployment** - Deploying my webpage with a PaaS (Vercel)
